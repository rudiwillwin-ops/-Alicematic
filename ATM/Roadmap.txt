Create a lightweight MetaTrader 5 Expert Advisor (MQL5).

BOT NAME:
NeutralGridLite

CORE LOGIC:
The bot trades a neutral grid strategy.

FEATURES:
- Start/Stop button on chart
- Places Buy Stop above current price
- Places Sell Stop below current price
- Grid distance configurable in pips
- Maximum grid levels configurable
- Fixed lot size input
- Basket profit target (close all trades)
- Equity drawdown protection
- Magic number system

INPUT PARAMETERS:

LotSize = 0.01
GridDistance = 20
MaxGridLevels = 5
BasketProfitUSD = 5
MaxDrawdownPercent = 20
MagicNumber = 55555

LOGIC:

1. When Start button pressed
   - Record current price
   - Place Buy Stop above price
   - Place Sell Stop below price

2. When price moves GridDistance
   - Add new grid order in that direction

3. Track total open trade profit.

4. If profit >= BasketProfitUSD
   - Close all positions
   - Reset grid

5. If equity drawdown >= MaxDrawdownPercent
   - Close all trades
   - Stop trading

UI:
Display on chart:
- Bot Status
- Current Profit
- Grid Level
- Equity Drawdown

The EA must compile with no errors and be optimized for low CPU usage.

Use OnTick event only
